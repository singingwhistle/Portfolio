Update: 07/19/17

Today I'm going to talk about the "core 4" in python; functions, loops, conditionals, and variables.
Functions "hold" a bit of code for you, defining an action you want to perform. For example: if you define a function "ask" to ask the user a question, then respond, instead of repeating the same lines of code over and over when you want to do that you can just run the function "ask".
Loops simply repeat a bit of code a certain number of times. It could be 3, until play == false, forever, or as long as you want.
Conditionals run different lines of code under different circumstances. For example, if the user inputs "y", print "yes".
Variables hold a number. they are used when you want a number to change, with the changes having influences on your code.
